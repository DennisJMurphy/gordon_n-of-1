# Iteration 10 — Revisions

## 1. Include standard interventions in baseline
In both the reports and the baseline editor
standard interventions are missing
the user should be able to set which supplements or activities they always use
eg: NMN mornings sublingual, TMG mornings and afternoons with food, like in the intervention screen.
each of these baseline interventions should also have a start-date attribute
eg: NMN since dd-mmm-yyyy or mmm-yyyy with consistency in rough % (99%)
explanation: this self-experimentation is the reason for the app, we need to show the reference of how long and how consistently habitual supplementation and habits have been used

## 2. NMN and TMG
the note about TMG should only be present if NMN is being supplemented, or not at all.

## 3. Other interventions
it should be possible for the user to add other interventions or habits not already listed
these could be daily, monthly, quarterly interventions, in the baseline or in the episode intervention
