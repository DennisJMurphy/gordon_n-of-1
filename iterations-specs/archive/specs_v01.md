General specs (v0.1, intentionally high-level)

**Platform**

Mobile app (React Native)

Offline-first

Dark, calm visual language

**Data types**

Anonymous baseline context (bracketed)

Episodes (observational or intervention)

Passive signals (sleep, steps, etc.)

Weekly self-reported summaries

Optional notes

**Interaction cadence**

First launch: ~2–3 minutes

Weekly check-in: <60 seconds

Episode close: ~2 minutes

**Outputs**

Human-readable episode summary

Machine-readable export (CSV / JSON)

Clear metadata about uncertainty & scope

**Integrations (export targets)**

Open Humans

Rejuve.ai

Alethios

Local file system
